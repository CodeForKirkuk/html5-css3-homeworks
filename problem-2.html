9. How to Submit your Solution
        a. Download **(clone)** this project `git clone https://github.com/CodeForKirkuk/html5-css3-homeworks`
        b. Then change your directory into downloaded project using `cd html5-css3-homeworks`
        c. Create new **branch** and name it with your fullname using `git checkout -b `
        d. Create a new html file for each **homework / problem** for example:
            * problem-1.html
            * problem-2.html
            * problem-3.html
        e. Then `git add` and `git commit` to commit to your code
        f. Push your **branch** up to GitHub `git push -u origin your-full-name`. That's it ;)
